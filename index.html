⚙️ 還沒設定 Firebase

請依 README.md 建一個 Firebase 專案，把網頁設定貼到 public/firebase-config.js，重新整理即可。

  1. 到 console.firebase.google.com 建立專案
  2. 建立 Firestore Database（正式或測試模式皆可，規則用 README 提供的）
  3. 啟用 Authentication → 匿名登入
  4. 專案設定 → 你的應用程式 → 複製 firebaseConfig 貼進 firebase-config.js
